# Prism

Prism 是一个面向国内用户的纯中文远程终端控制方案：

- QQ 负责消息通道
- KIMI Code 负责智能 CLI
- 安装目标是小白用户双击即可走完

## 卖点

- 纯中文安装引导
- QQ 直接对话，不需要命令行基础
- KIMI Code 本地接入，适合国产方案
- 安装完成后自动生成 `安装说明.txt`

## 安装

直接双击 `install.bat`。

安装器会尽量自动完成环境准备，你只需要：

1. 登录 KIMI
2. 登录 NapCat
3. 填两个 QQ 号

## 重新配置

如果只是换 QQ 号，双击 `setup.bat`。

## 启动

双击 `start.bat`。

## 目录简化说明

当前项目已经尽量压缩为这些核心入口：

- `install.bat`：首次安装
- `setup.bat`：重新配置
- `start.bat`：日常启动
- `napcat-start.bat`：单独拉起 NapCat
- `src/installer.ts`：唯一安装向导

## 常见问题

### QQ 每次先冒出一条 `????????????`

这通常不是 KIMI 的问题，而是 OneBot 发送层把文本包成了消息段数组，NapCat/QQ 客户端出现了兼容性异常。现在已经改成直接发送纯文本消息。

### 安装后看说明文档在哪

安装结束会自动打开 `安装说明.txt`，文件也会保存在项目根目录。
