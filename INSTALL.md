# Prism 安装说明

`Prism` 是面向国内中文用户的 QQ 远程终端控制方案：

- IM 层：QQ / NapCat
- 智能 CLI：KIMI Code
- 目标：纯中文、小白可用、一键安装

## 最简流程

1. 双击 `install.bat`
2. 按提示登录 `KIMI`
3. 按提示登录 `NapCat`
4. 填两个 QQ 号
5. 双击 `start.bat`

安装完成后，程序会自动生成并打开 `安装说明.txt`。

## 重新配置

如果只想重填 QQ 号，双击 `setup.bat` 即可。

## 日常启动

1. 双击 `start.bat`
2. 保持 NapCat 在线
3. 在你自己的 QQ 给机器人 QQ 发消息
